-------Add submodules(another git repo as submodule)to current repo in github--------------------------

1. git init in current folder
2. Clone the current repo 
Ex: git clone https://github.com/Praveenraj3537/GitExperiment.git --recurse-submodules
3. Go inside cloned folder
Ex: cd /GitExperiment
4. add submodule to the current repo
Ex: git submodule add https://github.com/Praveenraj3537/finalproject.git
5. Add add the files/folders
Ex: git add .
6. Commit to the repo and add message
Ex: git commit(add message)
7. Push to repo in github
Ex: git push (Check in the original link is it updated?)


Note: In clone if submodules files are not updating the use: git submodule update --init --recursive
